### 1. What is Static File Serving
- Returning files that already exist, such as HTML, CSS, JS, images, or fonts, in response to a clientâ€™s request, without any additional logic or database access.

||Static File|Dynamic File|
|--|--|--|
|Creation|Already exists as a file|Generated dynamically when requested|

### 2. Why Static Serving is Important?
- Reduces the server's burden
- Enhances response speed
- Improves sequrity
- Saves cost
